Manage Overtime Requests from Employee Odoo App

Manage Overtime Request in Odoo

This app allows your employees to make their own overtime request and also allow you to create multiple overtime request as shown and explained below.

Main Features:

  • Options
    • Option 1 - Overtime Requests: This will be requested by the employee for their own overtime request.
    • Option 2 - Multiple Overtime Requests: This request can be created and managed by the HR Manager or Department Manager on behalf of multiple employees together.
  • This module adds a new group called "Department Manager (Overtime)" on users' form settings.
  • Employees can create their own overtime requests and confirm them which will go through the approval process as shown.
  • Employee HR Officer / Manager (Administrator) can approve overtime requests from employees.
  • Employee HR Officer / Manager (Administrator) can create multiple overtime requests for multiple employees as shown.
  • Department managers can approve overtime requests from employees.
  • Department Manager employees can approve multiple overtime requests as shown.
  • For more details, please check the video and below screenshots.

Employee Overtime Request Approval by Employees/Users Entities.

  • Employee
  • HR Officer / Manager
  • Department Manager

Role of Employee

  • Employees can create and manage their own overtime requests and confirm them for the approval process.

Role of HR Officer / Manager(Administrator)

  • Employee HR Officer / Manager (Administrator) can approve overtime requests.
  • Employee HR Officer / Manager (Administrator) can create multiple overtime requests for multiple employees.

Role of Department Manager

  • Department Managers can approve overtime requests from employees.
  • Department Manager can approve multiple overtime requests/group overtime requests.
